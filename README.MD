# 🏦 Sistema Bancário em Python

Projeto desenvolvido em **Python** utilizando os conceitos de **Programação Orientada a Objetos (POO)** para simular um sistema bancário através de uma interface em linha de comando (CLI).

Este projeto faz parte da minha jornada de aprendizado em Python e tem como objetivo aplicar boas práticas de desenvolvimento, organização de código e modelagem orientada a objetos.

---

# Objetivos do Projeto

Este projeto foi desenvolvido para praticar conceitos fundamentais de Programação Orientada a Objetos, como:

- Classes e Objetos
- Encapsulamento
- Modularização
- Separação de responsabilidades
- Organização de projetos Python
- Tratamento de exceções
- Lógica de negócio

---

# Funcionalidades

O sistema permite realizar operações bancárias como:

- Cadastro de clientes
- Criação de contas bancárias
- Depósitos
- Saques
- Consulta de saldo
- Gerenciamento das operações pelo banco
- Tratamento de exceções personalizadas

---

# Tecnologias Utilizadas

- Python 3.11+
- Programação Orientada a Objetos (POO)

# Organização dos Módulos

### `entities/`

Contém as entidades principais do sistema.

- **Cliente**: representa um cliente do banco.
- **Conta**: representa uma conta bancária e suas operações.

---

### `service/`

Contém a lógica de negócio da aplicação.

- **Banco**: responsável pelo gerenciamento dos clientes, contas e operações bancárias.

---

### `utils/`

Contém recursos auxiliares utilizados pelo sistema.

- Exceções personalizadas.
- Classes e utilitários de apoio.

---

###  `dsa_mini_projeto2.py`

Arquivo principal da aplicação.

É responsável pela execução do sistema e pela interface de linha de comando (CLI), permitindo a interação do usuário com as funcionalidades disponíveis.
